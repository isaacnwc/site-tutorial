Coluna - DIV é tag neutra

A DIV ela é genérica, podendo ser "substituida" por tags semânticas!
Por exemplo, o TOPO pode ter sua tag -div- substituida por -HEADER-, as colunas podem ter sua tag -div- substituida por -section-
O menu pode ter sua tag DIV generica substituida por -nav-
Caso houvesse um rodapé na pagina, o div dele poderia ser substituido por -footer-
DIVS E TAGS SEMANTICAS SÃO ESTILIZADAS COM CSS, elas não tem um visual próprio

Principais tags semanticas

article: Define conteúdo independente e autônomo.
aside: Define conteúdo à parte do conteúdo da página.
details: Define detalhes adicionais que o usuário pode visualizar ou ocultar.
figcaption: Define uma legenda para um elemento --.
figure: Especifica conteúdo autônomo, como ilustrações, diagramas, fotos, listagens de código, etc.
footer: Define um rodapé para um documento ou seção.
header: Especifica um cabeçalho para um documento ou seção.
main: Especifica o conteúdo principal de um documento. mark: Define texto marcado/destacado.
nav: Define links de navegação.
section: Define uma seção em um documento.
summary: Define um cabeçalho visível para um elemento -details-.
time: Define uma data/hora.

Coluna - TAG Semantica

É interessante usar a TAG semantica para melhorar o entendimento do site, uma pessoa que pega o seu código para ler consegue entender bem melhor uma tag já estabelecida a uma tag genérica
Com div, é possível construir qualquer tipo de conteúdo, e ela é bem utilizada, porém deve saber julgar se a tag continuará sendo -div- ou se há alguma tag semantica para deixar mais claro o entendimento
