Imagens

É possível inserir imagens de duas maneiras: Usando um arquivo local da maquina ou uma imagem online já hospedada

-img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcQulnOkcc-hv8l4hKhKvk8Jqme9YRtGeESHOg&s"

Essa imagem está ligada a outro site

Para colocar uma imagem direto do seu PC, será necessário criar um novo diretório na pasta onde está seu site

-img src="img/jonkler.png"

É possível ajustar as dimensões da imagem usando WIDTH(Largura) e HEIGHT(Altura)

-img src="img/jonkler.png" width="300" height="300"

iFrames

Permite a inclusão de outro HTML dentro do primeiro, a partir da pagina inicial é possivel abrir uma seção de outra página qualuer

-iframe src="CSS.HTML" title="teste123">Meu iframe-/iframe/

Para colocar um video do youtube para o iframe, deverá clicar com o botão direito e copiar incorporação

iFrames também podem ter suas dimensões alteradas usando WIDTH e HEIGHT
